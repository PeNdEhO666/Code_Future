# Random Task Generator

**Автор:** Ломтев Антон

## Описание

Random Task Generator — это GUI-приложение на Python, которое помогает генерировать случайные задачи для повышения продуктивности. Приложение позволяет:

- Генерировать случайные задачи из предопределённого списка
- Добавлять свои собственные задачи с указанием типа
- Фильтровать историю задач по типу (учёба, спорт, работа)
- Сохранять историю задач в JSON файл и загружать её при запуске
- Очищать историю

## Требования

- Python 3.6+
- Стандартные библиотеки: tkinter, random, json, os, datetime
