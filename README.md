# Training Planner

**Автор:** Ломтев Антон

## Описание

Training Planner — это графическое приложение на Python для планирования и учёта тренировок. Приложение позволяет:

- Добавлять тренировки с указанием даты, типа и длительности
- Просматривать список всех тренировок в таблице
- Фильтровать тренировки по типу и дате
- Удалять тренировки
- Автоматически сохранять данные в JSON-файл
- Загружать данные при запуске приложения

## Требования

- Python 3.7+
- Стандартные библиотеки Python (tkinter, json, datetime, os)
